# SplitSecond MVP 💸

## Overview
SplitSecond is a FinTech app that allows users to quickly split bills based on individual consumption.

Instead of splitting evenly, users can:
- Assign items to specific people
- Share items across multiple participants
- Automatically calculate how much each person owes

## Features
- Add participants
- Add items with prices
- Assign items to individuals or groups
- Automatic bill calculation
- Simple and intuitive interface

## How it works
1. Enter participants
2. Add items and prices
3. Assign who consumed each item
4. The app calculates the exact amount per person
